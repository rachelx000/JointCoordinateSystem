### Local Deployment
1. Clone this repo: 
    ```
    git clone git@github.com:rachelx000/JointCoordinateSystem.git
    cd JointCoordinateSystem
    ```
2. Install all required packages and dependencies:
    ```
    npm install
    ```
3. Run the Vite development server to run the app locally:
    ```
    npx vite
    ```
